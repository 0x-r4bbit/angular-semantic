accordion (directive in module ui.semantic.accordion )

Description

An accordion displays a single piece of content, while allowing the option of displaying other related content.

Events There are 3 events which evaluate expressions used by the accordion directive: - onClose() - onOpen() - onChange()

Usage

as element:
<accordion
       exclusive="{boolean=true}"
       collapsible="{boolean=true}"
       duration="{number=400}"
       easing="{string=linear}"
       on-close="{expression}"
       on-cpen="{expression}"
       on-change="{expression}">
</accordion>
as attribute
<ANY accordion
     exclusive="{boolean=true}"
     collapsible="{boolean=true}"
     duration="{number=400}"
     easing="{string=linear}"
     on-close="{expression}"
     on-cpen="{expression}"
     on-change="{expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

  • exclusive – {boolean=true} –

    Set to false to allow multiple sections to be open at the same time

  • collapsible – {boolean=true} –

    Set to false to require an accordion to always have a section open

  • duration – {number=400} –

    Duration in ms of opening animation

  • easing – {string=linear} –

    Easing equation used for accordion (additional options require jQuery easing

  • on-close – {expression} –

    AngularJS expression to be evaluated when a content element is closed.

  • on-cpen – {expression} –

    AngularJS expression to be evaluated when a content element is opened.

  • on-change – {expression} –

    AngularJS expression to be evaluated when a content element is changed.
