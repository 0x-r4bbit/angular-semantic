toggle (directive in module ui.semantic.checkbox )

Description

A checkbox visually indicates the status of a user's selection

Events

There are 3 events which evaluate expressions used by the toggle directive:

  • onDisable(checked)
    • checked {Boolean} The checkbox's checked state
  • onEnable(checked)
    • checked {Boolean} The checkbox's checked state
  • onChange(checked)
    • checked {Boolean} The checkbox's checked state

Usage

as element:
<toggle
       on-enable="{expression}"
       on-disable="{expression}"
       on-change="{expression}">
</toggle>
as attribute
<ANY toggle
     on-enable="{expression}"
     on-disable="{expression}"
     on-change="{expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

  • on-enable – {expression} –

    Callback after a checkbox is enabled.

  • on-disable – {expression} –

    Callback after a checkbox is disabled.

  • on-change – {expression} –

    Callback after a checkbox is either disabled or enabled.
