radio (directive in module ui.semantic.checkbox )

Description

A radio button is an exclusively selectable option of a set.

The radio button must be a child of a radio-group.

Events

  • onSelect(selected, value)
    • selected {JQuery element} The selected radio button
    • value {*} The selected radio button's model value

Usage

as element:
<radio
       value="{string}"
       on-select="{expression}">
</radio>
as attribute
<ANY radio
     value="{string}"
     on-select="{expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

  • value – {string} –

    The model value of the radio button.

  • on-select – {expression} –

    Fires after the item is selected.
