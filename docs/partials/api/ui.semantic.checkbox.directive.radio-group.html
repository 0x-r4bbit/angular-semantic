radio-group (directive in module ui.semantic.checkbox )

Description

A radio-group is a set of mutually exclusive selectable options. The group itself holds a reference to a model, which is manipulated by the selection of child radio elements.

The radio-group element requires a name attribute, which each child radio element will automatically share.

Events

  • onSelect(selected, value)
    • selected {JQuery element} The selected radio button
    • value {*} The selected radio button's model value

Usage

as element:
<radio-group
       name="{string}"
       ng-model="{string}"
       on-select="{expression}">
</radio-group>
as attribute
<ANY radio-group
     name="{string}"
     ng-model="{string}"
     on-select="{expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

  • name – {string} –

    required The name of the form field.

  • ng-model – {string} –

    required The data binding for the radio buttons

  • on-select – {expression} –

    Callback after a checkbox is enabled.
